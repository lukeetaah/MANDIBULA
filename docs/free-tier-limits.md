# Límites gratuitos

Los límites cambian; verificar los paneles oficiales antes de publicar.

- Supabase: el plan gratuito puede pausar proyectos inactivos y limita base, egreso, conexiones y Realtime. El diseño agrupa comandos, conserva pocos snapshots y evita presencia por agente.
- Vercel: el plan Hobby limita uso comercial y recursos de funciones/banda. La simulación local evita CPU persistente del servidor.
- GitHub Actions: minutos y almacenamiento dependen de visibilidad/plan. Cachear pnpm y no descargar navegadores en cada job cuando la imagen ya los incluye.
- Navegador: WebGL, memoria y batería son límites del cliente; se ofrece DPR acotado y objetivo inicial de 150 agentes.

No existe API paga ni modelo de IA durante la partida. Antes de escalar, instrumentar bytes por match, mensajes por segundo, snapshot medio y duración real.
