# Recursos y licencias

El vertical slice no incluye recursos externos.

- Modelos: geometría procedural generada en ejecución con Three.js.
- Terreno y materiales: colores y formas del código, sin texturas descargadas.
- Audio: ruido ambiental sintetizado en Web Audio; sin muestras grabadas.

Si se añade un asset, registrar nombre, autor, URL original, licencia, modificaciones y archivo que lo usa. Se prefieren CC0 y OFL.
