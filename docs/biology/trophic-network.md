# Red trófica

## Vocabulario

- Depredación: Vespula y arañas consumen artrópodos vivos o recién muertos.
- Carroñeo: Vespula y descomponedores consumen cadáveres.
- Herbivoría/fungicultura: Acromyrmex corta material vegetal para su cultivo; no “depreda hojas”.
- Polinización: Bombus obtiene néctar/polen y transporta polen accidentalmente.
- Detritivoría/xilofagia: Porotermes usa madera húmeda y materia vegetal muerta.
- Competencia: rutas, flores, refugios y material pueden solaparse sin consumo.
- Parasitoidismo: reservado para taxones verificados; no está simulado en el slice.
- Defensa: matar sin consumir no se registra como depredación.
- Ingeniería: nidos, basureros, telas y galerías cambian rutas.
- Alianza: abstracción externa; nunca diálogo ni tratado dentro del mundo.
